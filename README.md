# 🍳 The Recipe Box

A personal kitchen companion that helps you figure out what to cook based on what you already have at home.

## What it does

**Cook Now** — Add ingredients from your pantry and instantly see which recipes you can make, ranked by how many ingredients you already have.

**Recipes** — Browse the full recipe collection, filter by meal type and cooking time, and view step-by-step instructions.

**Weekly Planner** — Plan your meals for the entire week by assigning recipes to breakfast, lunch, and dinner slots.

**Shopping List** — Automatically generates a shopping list based on your weekly plan vs. what's already in your pantry. Check off items as you shop.

## Built with

- React 19
- Vite
- Tailwind CSS
- Lucide React
